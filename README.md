# Aurora Gift — English ⇄ Russian

This is a cute, mobile-friendly mini-site gift.

## Open it
- Open `index.html` in a browser.

## Quick personalization
You can edit these directly on the page:
- her name
- your name
- one memory

You can also open with URL parameters if you want:
- `?name=Anna&from=Fernando&memory=our%20first%20language%20joke`

## What is inside
- giant hero section
- loveable letter
- personality cards
- English/Russian phrase section
- Russian voice buttons
- quiz
- melody buttons
- compliment machine
- constellation messages
- star-catching mini game
- live personalized final note

Made to feel warm, sweet, and ready to share.
